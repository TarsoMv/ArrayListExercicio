# Sistema de Biblioteca — ArrayList Exercício

Aplicação console em Java para gerenciamento de um sistema de biblioteca. O projeto utiliza `ArrayList` para armazenar os livros e a interface `Comparable` para ordená-los automaticamente em ordem alfabética pelo título.

## Funcionalidades

-  **Adicionar** livros ao acervo
-  **Listar** todos os livros em ordem alfabética
-  **Buscar** um livro pelo título
-  **Remover** um livro pelo título



## Tecnologias utilizadas

- **Java** — Linguagem principal do projeto
- **ArrayList** — Estrutura de dados para armazenar os livros
- **Interface Comparable** — Ordenação dos livros por título

## Como executar

**Pré-requisitos:** Java JDK instalado (versão 8 ou superior)

1. Clone o repositório:
   ```bash
   git clone https://github.com/TarsoMv/ArrayListExercicio.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd ArrayListExercicio
   ```

3. Compile os arquivos Java:
   ```bash
   javac src/*.java -d out/
   ```

4. Execute a aplicação:
   ```bash
   java -cp out/ Main
   ```
